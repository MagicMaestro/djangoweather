About the AQI Project

Uses: Python and the Django Framework, HTML5, CSS3

This is a basic Python and Django project created to provide the Air Quality for a user-provided zip code.

This project is a personal development project, aimed at expanding on my knowledge of Python by utilizing and testing the Django framework to build a simple website.

In addition, the project also utilizes the AirNow API to obtain updated data.
